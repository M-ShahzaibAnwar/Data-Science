Pokémon Data Analysis | Python, Pandas, Matplotlib, Seaborn

Analyzed a structured Pokémon dataset to uncover offensive, defensive, and generational performance trends.

Performed data cleaning and exploratory data analysis (EDA) to handle missing values and understand feature distributions.

Computed statistical metrics (mean, median, max, min) across primary and secondary Pokémon types.

Visualized insights using bar charts and heatmaps to compare attack, defense, speed, HP, and total stats.

Identified key insights such as dominant offensive/defensive types and evolution of base stats across generations.
