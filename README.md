# Data-Science
# Data Science Projects Portfolio

Welcome to my **Data Science Projects** repository! This repository showcases a collection of projects demonstrating practical skills in **Python, Pandas, NumPy, Scikit-learn, Machine Learning, Data Visualization, Web Scraping, NLP, and Data Analysis**. Each project includes clean, reproducible code along with explanations of the problem, solution approach, and outcomes.

---

## **Projects Included**

1. **Pokémon Data Analysis** – Explore offensive, defensive, and generational stats using Pandas and visualizations.  
2. **Customer Churn Prediction** – Predict which customers are likely to leave using Logistic Regression and Random Forest.  
3. **Titanic Survival Prediction** – Classify passenger survival using historical data and feature engineering.  
4. **Holocaust History Articles Scraper** – Scrape and structure articles from Google search results using BeautifulSoup and Pandas.  
5. **Patient Selection Analysis** – Compare Roulette Wheel and Random selection strategies based on a calculated Fit_Score.  
6. **Social Media User Data Management** – Perform data operations (CRUD) and preprocessing using Pandas.  
7. **Student Performance Clustering** – Segment students into performance clusters using K-Means and visualize results.  
8. **Quora Question Text Cleaning & Feature Engineering** – Clean, stem, lemmatize, and extract features from question pairs for NLP tasks.  
9. **Diabetes Prediction** – Build and evaluate SVM and Logistic Regression models to predict diabetes risk.  

---

## **Key Skills & Techniques**
- **Programming & Libraries:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, BeautifulSoup, Regex, NLTK  
- **Data Handling:** Cleaning, preprocessing, normalization, feature engineering  
- **Machine Learning:** Classification, Regression, Clustering, Model Evaluation  
- **Data Visualization:** Bar charts, Heatmaps, Scatter plots, Histograms  
- **Web Scraping & Automation:** Article extraction from websites
